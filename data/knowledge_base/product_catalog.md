# Product Catalog and Pricing

Annual Contract Value (ACV): $20,000 - $60,000
Key Features:
- Up to 50 user seats
- Standard CRM and pipeline management
- Basic analytics and reporting
- Email and chat support (business hours)
- Standard API access (10,000 calls/day)

Best Fit Industries: Retail, logistics, e-commerce, professional services

### Enterprise Plan

Key Features:
- Unlimited user seats
- Advanced AI-powered analytics
- Custom workflow automation
- Dedicated Customer Success Manager
- 99.99% SLA uptime guarantee

- Priority 24/7 support

- SSO/SAML integration

Best Fit Industries: Financial services, healthcare, technology, media & entertainment

## Common Objections and Responses

### "Your pricing is too high"
Response: Focus on ROI. Our Enterprise customers see an average 3.2x return in 
year 1 through productivity gains and pipeline visibility. Request an ROI workshop 
with the account's finance team.

### "We already use [competitor]"
Response: Ask about their pain points with the current solution. Our differentiation 
is (1) AI-native analytics, (2) superior API ecosystem, and (3) white-glove 
onboarding that reduces time-to-value from months to weeks.

### "We need security/compliance certification X"
Response: We have SOC2 Type II, ISO 27001, HIPAA BAA available, and FedRAMP in 
progress. Connect them with our Security team for a detailed compliance review.
